# ReKV Analysis

This repository is an **unofficial, analysis-focused** project **based on**

> "Streaming Video Question-Answering with In-context Video KV-Cache Retrieval", *ICLR* 2025.

The goal of this repo is to **reproduce, inspect, and analyze** ReKV-style streaming VideoQA and in-context video KV-cache retrieval for research purposes.

- Built on top of the official ReKV implementation and related public code.
- Contains custom scripts and configs for experiments and analysis only.
- **Not** an official implementation and **not** affiliated with or endorsed by the original authors.

For detailed instructions on setup, data preparation, and evaluation, please refer to the official ReKV repository: [ReKV Official Repo](https://github.com/Becomebright/ReKV).
